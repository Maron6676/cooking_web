# Cooking web

## サイト概要
### サイトテーマ
- 料理を細かくカテゴリー分けし、検索しやすい料理レビューサイト

### テーマを選んだ理由
- 料理のレシピを検索する際に、名前やジャンルでの検索しかできないサイトが多った。そこで、初心者向けなどのカテゴリーや調理時間など、さまざまなカテゴリーで検索できるような仕組みを導入したいと考えました。
- 

### ターゲットユーザ
- 料理する人

### 主な利用シーン
- 料理のレシピを探す時
- 家にある材料で作れるものを探す時
